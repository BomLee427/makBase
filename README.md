### 목록조회 : [GET] /makkoli
#### 필터
- 전통주 분류여부
- 도수
- 가격
- 스타일
- 지역
#### 검색
- 이름
#### 정렬
- 국문명
- 영문명
- 도수
- 산미
- 당도
- 아스파탐 첨가량
- 가격
#### 내용
- 국문명
- 영문명
- 지역

### 단일조회 : [GET] /makkoli/{id}
#### 내용
- 국문명

- 산미
- 당도

- 가격

- 스타일
